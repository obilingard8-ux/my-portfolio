# Obi Chibueze Vincent - Portfolio

A modern, responsive portfolio website showcasing my work as a Frontend Developer, UI/UX Designer, and Graphic Designer.

## Features

- **Modern Design**: Clean, professional design with smooth animations
- **Fully Responsive**: Works seamlessly on all devices
- **Dark Theme**: Elegant dark theme with gradient accents
- **Project Filtering**: Filter projects by category
- **Contact Form**: Ready-to-use contact form
- **Performance Optimized**: Fast loading and smooth interactions

## Sections

1. **Hero**: Introduction with call-to-action
2. **About**: Personal story and highlights
3. **Projects**: Showcase of work with filtering
4. **Skills**: Technical skills with progress indicators
5. **Contact**: Contact form and social links
6. **Footer**: Navigation and copyright

## Tech Stack

- React 18
- JavaScript (ES6+)
- CSS3 (Custom Properties, Flexbox, Grid)
- React Icons
- Framer Motion (for animations)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

## Customization

### Adding Your Photos

Replace the placeholder elements in:
- `Hero.js` - Main profile photo
- `About.js` - About section photo

### Updating Contact Information

Edit the `Contact.js` file to update:
- Email address
- Phone number
- Location
- Social media links

### Adding Projects

Edit the `projects` array in `Projects.js` to add your own projects with:
- Title
- Description
- Category (frontend, design, graphics)
- Technologies used
- Live URL
- GitHub URL (optional)

### Updating Skills

Modify the `skillCategories` array in `Skills.js` to reflect your actual skill levels.
